1.安装ruby环境
2.安装sass  gem install sass
3.安装grunt npm install -g grunt
4.进入grunt目录，安装node模块 npm install
5.编译sass  grunt
6.如果使用compass，继续往下，安装compass  gem install compass
7.进入compass/grunt目录，安装node模块 npm install
8.执行compass grunt
